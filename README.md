## Development

1. Ensure you have [yarn](https://yarnpkg.com/en/docs/install#mac-stable) installed.

2. Install dependencies:

   ```
   $ yarn install
   ```

3. Start development environment:

   ```
   $ yarn start
   $ yarn dev:desktop
   ```
